# **App Name**: Code Gemini

## Core Features:

- Real-time Suggestions: Stream real-time code suggestions powered by Gemini as the user types, improving coding efficiency.
- Code Autocorrection: Automatically correct code snippets using Gemini, improving code quality.
- Code Generation: Generate code snippets from natural language prompts using Gemini as a tool, expediting development.
- Firebase Integration: Securely connect to Firebase for authentication and to call backend functions for AI processing.
- Language Support: Support multiple languages including JavaScript, Python, TypeScript, Java, C++, SQL, and C# (.NET).
- Light/Dark Mode Toggle: Implement a toggle for switching between light and dark themes in the popup UI, improving user experience.
- Internal Testing Controls: Enable internal testing banner and employee email allow-list, restricting access during initial rollout.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calm and reliable coding atmosphere.
- Background color: Light gray (#F0F4F8) for the light theme, dark gray (#282C34) for the dark theme, providing comfortable contrast.
- Accent color: Pale cyan (#B0E57C), for focus and engagement in function feedback.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-centric feel.
- Body font: 'Inter' (sans-serif) to improve readability.
- Code font: 'Source Code Pro' (monospace) to format code snippets distinctly.
- Use minimalist icons for settings and actions in the popup UI.
